# Tools

Part of the [Krypton Toolkit Suite Extended](https://github.com/Krypton-Suite/Extended-Toolkit) packages.

## Description

This package is an extension to the [Standard Krypton Toolkit](https://github.com/Krypton-Suite/Standard-Toolkit), providing additional controls and utilities for Windows Forms applications.

## Installation

Install via NuGet Package Manager:

```
Install-Package Krypton.Toolkit.Suite.Extended.Tools
```

Or using .NET CLI:

```
dotnet add package Krypton.Toolkit.Suite.Extended.Tools
```

## Framework Support

- .NET Framework 4.7.2 - 4.8.1
- .NET 8 - 10
